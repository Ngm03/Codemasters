Описание Проекта
Aqmola Start — это единая цифровая экосистема для Акмолинской области, объединяющая стартапы, IT-специалистов, инвесторов и организаторов мероприятий. Приложение решает проблему фрагментации информации в регионе, предоставляя удобный инструмент для нетворкинга и поиска возможностей.

Ключевые возможности:
- Каталог стартапов: Поиск и фильтрация инновационных проектов.
- Поиск команды: Платформа для объединения специалистов в команды.
- Афиша событий: Актуальная информация о хакатонах, митапах и конференциях.
- Рынок вакансий: Интеграция с HH.ru для поиска работы и сотрудников.
- Админ-панель: Инструменты модерации контента и управления пользователями.

Стек Технологий
- Frontend: Flutter (Dart) — кроссплатформенная разработка.
- Backend: Node.js (Express) — REST API.
- База данных: MySQL — хранение данных пользователей и контента.
- AI: Google Gemini — интеллектуальный ассистент.
- Интеграции: HH.ru API, Google Events, Product Hunt API.

Функционал
1. Ролевая модель: Разделение на разработчиков, фаундеров и инвесторов.
2. UGC (User Generated Content): Пользователи могут самостоятельно создавать карточки стартапов, вакансий и событий.
3. Модерация: Двухуровневая система проверки контента администраторами.
4. Локализация: Полная поддержка казахского, русского и английского языков.
5. Умный помощник: Чат-бот для навигации по экосистеме.

Установка и Запуск

Требования
- Flutter SDK (3.0.0+)
- Node.js (16+)
- MySQL Server

1. Настройка Бэкенда
1. Перейдите в папку backend:
   cd backend
2. Установите зависимости:
   npm install
3. Настройте подключение к БД в database.js.
4. Запустите сервер:
   node server.js

2. Запуск Приложения
1. Вернитесь в корень проекта:
   cd ..
2. Загрузите зависимости Flutter:
   flutter pub get
3. Запустите приложение:
   flutter run
4. Сборка APK (Release):
   flutter build apk

Использование AI и Внешних Инструментов (п. 5.4)
- Google Gemini API: Используется в модуле "NGM Assistant" для генерации ответов на вопросы пользователей о стартап-экосистеме.
- Flutter SVG: Для работы с векторной графикой.
- Shared Preferences: Для локального хранения настроек языка.
